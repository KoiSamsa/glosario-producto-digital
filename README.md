# Glosario de Producto, UX/UI e IA

Un glosario práctico de diseño de producto: **254 términos** con definición, cómo se hace o se calcula, un ejemplo y las trampas que nadie te cuenta. Pensado para que diseño, desarrollo, negocio y cualquier curiosa hablen el mismo idioma.

🔗 **[Abrir el glosario](https://koisamsa.github.io/glosario-producto-digital/)**
_(disponible una vez activado GitHub Pages en Settings → Pages)_

---

## Qué es

Una única página web, sin instalación ni servidor. Se abre en cualquier navegador y funciona sin conexión una vez cargada. Incluye:

- **Buscador instantáneo** — filtra según escribes, ignora acentos (busca `metrica` y encuentra `métrica`).
- **Filtros por categoría** — 9 áreas (en español): Proceso, Investigación, Interfaz, Producto, Crecimiento, Métricas, Sistemas de diseño, Tecnología y Accesibilidad.
- **Términos enlazados** — desde cada ficha saltas a los conceptos relacionados.
- **Enlaces directos** — cada término tiene su URL (`…#handoff`), compartible.
- **Accesible** — navegación por teclado, foco visible, contraste cuidado.
