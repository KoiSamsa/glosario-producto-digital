# Glosario de Producto, UX/UI e IA

Un glosario práctico de diseño de producto: **112 términos** con definición, cómo se hace o se calcula, un ejemplo y las trampas que nadie te cuenta. Pensado para que diseño, desarrollo, negocio y cualquier curiosa hablen el mismo idioma.

🔗 **[Abrir el glosario](https://koisamsa.github.io/glosario-producto-digital/)**
_(disponible una vez activado GitHub Pages en Settings → Pages)_

---

## Qué es

Una única página web, sin instalación ni servidor. Se abre en cualquier navegador y funciona sin conexión una vez cargada. Incluye:

- **Buscador instantáneo** — filtra según escribes, ignora acentos (busca `metrica` y encuentra `métrica`).
- **Filtros por categoría** — 8 áreas: UX Research, UI & Visual, Design Systems, Accesibilidad, Métricas, Product & Delivery, Métodos & Procesos, Técnico & IA.
- **Términos enlazados** — desde cada ficha saltas a los conceptos relacionados.
- **Enlaces directos** — cada término tiene su URL (`…#handoff`), compartible.
- **Accesible** — navegación por teclado, foco visible, contraste cuidado.

## Cómo usarlo

Solo abre el enlace de arriba. Para tenerlo en local: descarga `index.html` y ábrelo con doble clic.

## Diseño

La interfaz usa el **sistema visual de KOI SAMSA** (tipografía IBM Plex, paleta cromática de 11 piezas materiales). El glosario es, además, una pieza de ese sistema.
